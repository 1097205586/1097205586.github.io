记一个centos上安装mysql8.0.13的坑

发表于 2018-09-14

yum安装 mysql

1.yum仓库下载MySQL：sudo yum localinstall https://repo.mysql.com//mysql80-community-release-el7-1.noarch.rpm
2.yum仓库下载MySQL：sudo yum localinstall https://repo.mysql.com//mysql80-community-release-el7-1.noarch.rpm
3.yum仓库下载MySQL：sudo yum localinstall https://repo.mysql.com//mysql80-community-release-el7-1.noarch.rpm
4.检查MySQL服务状态：sudo service mysqld status
5.检查MySQL服务状态：sudo service mysqld status(这里非常重要一定要记下密码!!!)
6.本地MySQL客户端登录：mysql -uroot -p
7.输入密码为第5步查出的
8.修改root登录密码：ALTER USER ‘root‘@’localhost’ IDENTIFIED BY ‘密码’; 密码要数字大小写字母加符号不能小于8为数否则报错
9.修改端口号 /etc/my.conf port = 2018
10.配置外网访问 update user set host = ‘%’ where user =’root’;
11.show global variables like ‘port’;可查看MySQL服务端口，如果看到的value为0，则说明没有使用密码登录，需要去修改my.cnf文件；

错误解决 Host ‘XXX’ is not allowed to connect to this MySQL server

多数都是没有开启外网访问, update user set host = '%' where user ='用户名'; 把响应的用户外网访问权限加上,另外要查询3306端口号是否对外开放

错误解决 Access denied for user ‘root‘@’localhost’

修改 /etc/my.conf  添加skip-grant-tables 开启无密码登录 然后重启
进入 mysql 数据库 修改登录密码：ALTER USER 'root'@'localhost' IDENTIFIED BY '密码'; ***
